# Task Tracker

A full-stack CRUD app with a React frontend and a custom Spring Boot backend API.

## Features
- Create, read, update, delete tasks
- Set task status, priority, and due date
- Inline task editing
- Dark/light mode toggle
- Full-screen responsive layout
